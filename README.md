# BXP-SecretSonar v0.1.0-alpha

Universal proactive secret detection engine for all environments.

## Quick Start
```bash
pip install -e ".[dev]"
secretsonar --help
secretsonar info
pytest tests/unit/ -v
```

## Portability
Works natively on Linux, macOS, Windows, Termux, PRoot, iOS (iSH), and GitHub Codespaces without native dependencies.
